## 📊 현재 실력 진단

### 자가 진단 체크리스트

```
레벨 1: 입문 (0-3개월 경험)
□ Docker 개념을 이해한다
□ 기본 명령어를 사용할 수 있다
□ Dockerfile을 작성할 수 있다
□ docker-compose.yml을 이해한다
□ 간단한 컨테이너를 실행할 수 있다

레벨 2: 초급 (3-6개월 경험)
□ 멀티 컨테이너 앱을 구성할 수 있다
□ 네트워크와 볼륨을 설정할 수 있다
□ 이미지를 최적화할 수 있다
□ 기본 트러블슈팅이 가능하다
□ 개발 환경을 Docker로 구성할 수 있다

레벨 3: 중급 (6-12개월 경험)
□ CI/CD 파이프라인을 구축할 수 있다
□ 프로덕션 환경을 고려한 설계를 한다
□ 보안 모범 사례를 적용한다
□ 모니터링과 로깅을 설정할 수 있다
□ Docker Swarm을 사용할 수 있다

레벨 4: 고급 (1-2년 경험)
□ Kubernetes를 운영할 수 있다
□ 대규모 시스템을 설계할 수 있다
□ Infrastructure as Code를 구현한다
□ Service Mesh를 적용할 수 있다
□ 성능 튜닝을 할 수 있다

레벨 5: 전문가 (2년 이상 경험)
□ 복잡한 아키텍처를 설계한다
□ 팀을 리드하고 멘토링한다

□ 오픈소스에 기여한다
□ 비즈니스 가치를 창출한다

### 3. 페이만 학습법 (Feynman Technique)

```
1. 개념 선택
   예: "Docker 네트워크"

2. 설명하기 (블로그/영상)
   - 초등학생도 이해할 수 있게
   - 비유와 예제 사용

3. 막히는 부분 찾기
   - 설명하다 막히는 곳 = 모르는 부분

4. 다시 학습
   - 막힌 부분 집중 학습
   - 다시 설명

5. 단순화
   - 복잡한 용어 제거
   - 핵심만 남기기
```

### 4. 포모도로 테크닉

```
