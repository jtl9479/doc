## 🎯 메서드 오버라이딩이란? {#메서드-오버라이딩이란}

### 정의
메서드 오버라이딩(Method Overriding)은 **부모 클래스에서 정의된 메서드를 자식 클래스에서 재정의**하는 것입니다.

### 핵심 개념
```java
class Parent {
    public void display() {
        System.out.println("부모 클래스의 display()");
    }
}

class Child extends Parent {
    @Override  // 오버라이딩 어노테이션
    public void display() {
        System.out.println("자식 클래스의 display()");
    }
}

// 사용
Parent p = new Child();
p.display();  // "자식 클래스의 display()" 출력
// → 참조 변수는 Parent지만, 실제 객체는 Child이므로
// → Child의 display()가 호출됨 (동적 바인딩)
```

### 오버라이딩 vs 오버로딩

| 구분 | 오버라이딩 (Overriding) | 오버로딩 (Overloading) |
|------|------------------------|----------------------|
| 정의 | 부모 메서드를 자식이 재정의 | 같은 이름, 다른 매개변수 |
| 관계 | 상속 관계 필요 | 같은 클래스 내 |
| 메서드 시그니처 | 동일해야 함 | 달라야 함 |
| 바인딩 | 동적 바인딩 (런타임) | 정적 바인딩 (컴파일타임) |
| 어노테이션 | @Override | 없음 |

### 오버라이딩 규칙
1. **메서드 시그니처 동일**: 이름, 매개변수 타입, 개수가 같아야 함
2. **리턴 타입**: 같거나 하위 타입 (공변 반환 타입)
3. **접근 제어자**: 같거나 더 넓은 범위로만 변경 가능
4. **예외**: 더 많은 checked 예외를 던질 수 없음

```java
class Parent {
    protected Object getData() throws IOException {
        return new Object();
    }
}

class Child extends Parent {
    // ✅ 올바른 오버라이딩
    @Override
    public String getData() {  // String은 Object의 하위 타입
        return "data";
    }
}
```

## 📱 실생활 비유 1: 스마트폰 알림 시스템 {#비유1-스마트폰-알림}

### 비유 설명
모든 스마트폰은 알림을 보여주지만, 제조사마다 알림을 표시하는 방식이 다릅니다. 삼성은 엣지 라이팅, 애플은 배너 알림, 샤오미는 팝업 알림을 사용합니다.

### Java 코드로 구현

```java
/**
 * 모든 스마트폰의 부모 클래스
 */
public class Smartphone {
    protected String brand;
    protected String model;
    protected int batteryLevel;

    public Smartphone(String brand, String model) {
        this.brand = brand;
        this.model = model;
        this.batteryLevel = 100;
    }

    // 기본 알림 표시 메서드 (오버라이딩 대상)
    public void showNotification(String app, String message) {
        System.out.println("━━━━━━━━━━━━━━━━━━━━");
        System.out.println("📱 기본 알림");
        System.out.println(app + ": " + message);
        System.out.println("━━━━━━━━━━━━━━━━━━━━");
    }

    // 공통 메서드
    public void powerOn() {
        System.out.println(brand + " " + model + " 전원 ON");
    }

    public void showInfo() {
        System.out.println(brand + " " + model + " (배터리: " + batteryLevel + "%)");
    }
}

/**
 * 삼성 스마트폰 - 엣지 라이팅 알림
 */
public class SamsungPhone extends Smartphone {
    private String edgeLightingColor;

    public SamsungPhone(String model) {
        super("Samsung", model);
        this.edgeLightingColor = "파란색";
    }

    // ✅ 알림 메서드 오버라이딩
    @Override
    public void showNotification(String app, String message) {
        System.out.println("\n╔══════════════════════════╗");
        System.out.println("║   🌟 엣지 라이팅 알림    ║");
        System.out.println("╠══════════════════════════╣");
        System.out.println("║ 색상: " + edgeLightingColor + " 빛         ║");
        System.out.println("║ 앱: " + app + "              ║");
        System.out.println("║ 내용: " + message);
        System.out.println("╚══════════════════════════╝");

        // 엣지 라이팅 애니메이션
        animateEdgeLighting();
    }

    // 삼성 고유 기능
    private void animateEdgeLighting() {
        System.out.print("엣지 라이팅: ");
        for (int i = 0; i < 3; i++) {
            System.out.print("✨ ");
            try {
                Thread.sleep(200);
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }
        System.out.println();
    }

    public void setEdgeLightingColor(String color) {
        this.edgeLightingColor = color;
        System.out.println("엣지 라이팅 색상 변경: " + color);
    }

    // 삼성 전용 기능
    public void useSamsungPay() {
        System.out.println("💳 Samsung Pay 실행");
    }
}

/**
 * 애플 아이폰 - 배너 알림
 */
public class iPhone extends Smartphone {
    private boolean isDarkMode;

    public iPhone(String model) {
        super("Apple", model);
        this.isDarkMode = false;
    }

    // ✅ 알림 메서드 오버라이딩
    @Override
    public void showNotification(String app, String message) {
        String bgColor = isDarkMode ? "검정" : "흰색";
        String textColor = isDarkMode ? "흰색" : "검정";

        System.out.println("\n┌─────────────────────────┐");
        System.out.println("│  📲 배너 알림 (" + bgColor + ")  │");
        System.out.println("├─────────────────────────┤");
        System.out.println("│ " + app);
        System.out.println("│ " + message);
        System.out.println("└─────────────────────────┘");

        slideAnimation();
    }

    // 애플 고유 기능
    private void slideAnimation() {
        System.out.print("화면 상단에서 슬라이드: ");
        for (int i = 0; i < 5; i++) {
            System.out.print("↓ ");
        }
        System.out.println();
    }

    public void setDarkMode(boolean enabled) {
        this.isDarkMode = enabled;
        System.out.println("다크모드: " + (enabled ? "ON" : "OFF"));
    }

    // 애플 전용 기능
    public void useFaceID() {
        System.out.println("😊 Face ID 인증 중...");
    }

    public void useAirDrop() {
        System.out.println("📡 AirDrop 실행");
    }
}

/**
 * 샤오미 스마트폰 - 팝업 알림
 */
public class XiaomiPhone extends Smartphone {
    private boolean isFloatingMode;

    public XiaomiPhone(String model) {
        super("Xiaomi", model);
        this.isFloatingMode = true;
    }

    // ✅ 알림 메서드 오버라이딩
    @Override
    public void showNotification(String app, String message) {
        if (isFloatingMode) {
            System.out.println("\n╭───────────────────────╮");
            System.out.println("│  💬 플로팅 팝업 알림  │");
            System.out.println("├───────────────────────┤");
            System.out.println("│ 앱: " + app);
            System.out.println("│ 메시지: " + message);
            System.out.println("│ [답장] [무시]         │");
            System.out.println("╰───────────────────────╯");

            floatingAnimation();
        } else {
            super.showNotification(app, message);  // 부모의 기본 알림 사용
        }
    }

    private void floatingAnimation() {
        System.out.print("플로팅 모드: ");
        for (int i = 0; i < 3; i++) {
            System.out.print("🔵 ");
        }
        System.out.println();
    }

    public void setFloatingMode(boolean enabled) {
        this.isFloatingMode = enabled;
        System.out.println("플로팅 모드: " + (enabled ? "ON" : "OFF"));
    }

    // 샤오미 전용 기능
    public void useMiPay() {
        System.out.println("💰 Mi Pay 실행");
    }
}
```

3. **다형성**: `Smartphone` 타입으로 모든 폰을 관리하지만, 실제 동작은 각 객체의 타입에 따라 달라짐
4. **super 활용**: 샤오미는 조건에 따라 `super.showNotification()`으로 부모의 기본 알림 사용

## ☕ 실생활 비유 2: 커피 제조 방식 {#비유2-커피-제조}

### 비유 설명
모든 커피숍은 커피를 만들지만, 스타벅스, 이디야, 블루보틀은 각각 다른 방식으로 커피를 제조합니다.

### Java 코드로 구현

```java
/**
 * 커피숍 부모 클래스
 */
public class CoffeeShop {
    protected String name;
    protected int price;

    public CoffeeShop(String name, int price) {
        this.name = name;
        this.price = price;
    }

    // 기본 커피 제조 방법 (오버라이딩 대상)
    public void makeCoffee(String coffeeType) {
        System.out.println("\n━━━━━━━━━━━━━━━━━━━━");
        System.out.println("☕ 기본 커피 제조");
        System.out.println("━━━━━━━━━━━━━━━━━━━━");
        System.out.println("1. 원두 분쇄");
        System.out.println("2. 에스프레소 추출");
        System.out.println("3. 우유 스팀");
        System.out.println("4. 컵에 담기");
        System.out.println(coffeeType + " 완성!");
    }

    public void takeOrder(String customerName, String coffee) {
        System.out.println("\n📋 주문 접수");
        System.out.println("고객: " + customerName);
        System.out.println("주문: " + coffee);
        System.out.println("가격: " + price + "원");
    }
}

/**
 * 스타벅스 - 레시피 카드 시스템
 */
public class Starbucks extends CoffeeShop {
    private int recipeVersion;

    public Starbucks() {
        super("Starbucks", 5500);
        this.recipeVersion = 2024;
    }

    @Override
    public void makeCoffee(String coffeeType) {
        System.out.println("\n╔══════════════════════════╗");
        System.out.println("║  ⭐ Starbucks Coffee    ║");
        System.out.println("╠══════════════════════════╣");
        System.out.println("║ 레시피 버전: " + recipeVersion + "      ║");
```

```java
public class MSExcel extends DocumentEditor {
    private int rows;
    private int columns;
    private String[][] cells;

    public MSExcel(String fileName) {
        super(fileName + ".xlsx");
        this.rows = 100;
        this.columns = 26;
        this.cells = new String[rows][columns];
    }

    @Override
    public void save() {
        System.out.println("\n📊 ═══ MS Excel 저장 ═══");
        System.out.println("파일명: " + fileName);
        System.out.println("형식: Microsoft Excel 통합 문서 (.xlsx)");

        System.out.println("\n저장 과정:");
        System.out.println("1️⃣ 워크시트 분석");
        System.out.println("   - 시트 수: 3개 (Sheet1, Sheet2, Sheet3)");
        System.out.println("   - 행: " + rows);
        System.out.println("   - 열: " + columns);

        System.out.println("2️⃣ 셀 데이터 저장");
        saveCellData();

        System.out.println("3️⃣ 수식 저장");
        System.out.println("4️⃣ 차트 저장");
        System.out.println("5️⃣ 서식 저장");
        System.out.println("6️⃣ XML 형식으로 변환");
        System.out.println("7️⃣ .xlsx 파일 생성");

        isSaved = true;
        System.out.println("\n✅ 저장 완료!");
        System.out.println("💾 위치: C:\\Documents\\" + fileName);
        System.out.println("📊 파일 크기: " + calculateSize() + " KB");
    }

    private void saveCellData() {
        int filledCells = 0;
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < columns; j++) {
                if (cells[i][j] != null) {
                    filledCells++;
                }
            }
        }
        System.out.println("   - 입력된 셀: " + filledCells + "개");
    }

    private int calculateSize() {
        return rows * columns / 100;
    }

    public void setCell(int row, int col, String value) {
        if (row < rows && col < columns) {
            cells[row][col] = value;
            isSaved = false;
            System.out.println("셀 [" + row + "," + col + "] = " + value);
        }
    }

    public void insertFormula(int row, int col, String formula) {
        System.out.println("📐 수식 입력: [" + row + "," + col + "] = " + formula);
        isSaved = false;
    }
}
```

```java
        System.out.println("노랫소리 패턴 설정: " + pattern);
    }

    @Override
    public void showInfo() {
        super.showInfo();
        System.out.println("비행 가능: " + (canFly ? "예" : "아니오"));
        System.out.println("노랫소리: " + songPattern);
        System.out.println("━━━━━━━━━━━━━━━━━━━━");
```
