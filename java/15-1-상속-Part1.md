## 상속이란 무엇인가?

### 핵심 개념
상속(Inheritance)은 **기존 클래스의 필드와 메서드를 새로운 클래스가 물려받아 재사용하는 객체지향 프로그래밍의 핵심 개념**입니다.

### 기본 구조
```java
// 부모 클래스 (Super Class, Base Class)
public class Animal {
    String name;
    int age;

    public void eat() {
        System.out.println(name + "이(가) 먹습니다.");
    }

    public void sleep() {
        System.out.println(name + "이(가) 잡니다.");
    }
}

// 자식 클래스 (Sub Class, Derived Class)
public class Dog extends Animal {
    String breed;  // 자식만의 필드

    // 부모의 eat(), sleep() 자동 상속

    // 자식만의 메서드
    public void bark() {
        System.out.println(name + "이(가) 짖습니다: 멍멍!");
    }
}

// 사용
Dog dog = new Dog();
dog.name = "뽀삐";
dog.age = 3;
dog.eat();      // 부모로부터 상속받은 메서드
dog.sleep();    // 부모로부터 상속받은 메서드
dog.bark();     // 자식만의 메서드
```

### 상속의 특징

**1. extends 키워드 사용**
```java
public class 자식클래스 extends 부모클래스 {
    // 자식만의 필드와 메서드
}
```

**2. 단일 상속만 가능 (Java)**
```java
// ✅ 가능
public class Dog extends Animal { }

// ❌ 불가능 (다중 상속 불가)
public class Dog extends Animal, Pet { }  // 컴파일 에러

// ✅ 대신 인터페이스 사용
public class Dog extends Animal implements Pet, Companion { }
```

**3. super 키워드**
```java
public class Animal {
    String name;

    public Animal(String name) {
        this.name = name;
    }

    public void intro() {
        System.out.println("동물: " + name);
    }
}

public class Dog extends Animal {
    String breed;

    public Dog(String name, String breed) {
        super(name);  // 부모 생성자 호출
        this.breed = breed;
    }

    @Override
    public void intro() {
        super.intro();  // 부모 메서드 호출
        System.out.println("품종: " + breed);
    }
}
```

**4. 모든 클래스는 Object 상속**
```java
// 명시하지 않아도 자동으로 Object 상속
public class MyClass {
    // 실제로는: public class MyClass extends Object
}

// Object의 메서드 사용 가능
MyClass obj = new MyClass();
obj.toString();
obj.equals(null);
obj.hashCode();
```

### 상속의 장점

**1. 코드 재사용**
```java
// ❌ 상속 없이
public class Dog {
    String name;
    void eat() { }
    void sleep() { }
    void bark() { }
}

public class Cat {
    String name;
    void eat() { }      // 중복!
    void sleep() { }    // 중복!
    void meow() { }
}

// ✅ 상속 사용
public class Animal {
    String name;
    void eat() { }
    void sleep() { }
}

public class Dog extends Animal {
    void bark() { }  // 고유 기능만 추가
}

public class Cat extends Animal {
    void meow() { }  // 고유 기능만 추가
}
```

**2. 계층 구조 표현**
```java
Animal
  ├─ Mammal (포유류)
  │   ├─ Dog
  │   ├─ Cat
  │   └─ Human
  ├─ Bird (조류)
  │   ├─ Eagle
  │   └─ Penguin
  └─ Fish (어류)
      ├─ Shark
      └─ Goldfish
```

**3. 다형성 기반**
```java
Animal animal1 = new Dog();
Animal animal2 = new Cat();
Animal animal3 = new Bird();

// 배열로 관리 가능
Animal[] animals = {new Dog(), new Cat(), new Bird()};
for (Animal animal : animals) {
    animal.eat();  // 다형성
}
```

### 메모리 구조

```java
class Animal {
    String name = "동물";
    void eat() { }
}

class Dog extends Animal {
    String breed = "진돗개";
    void bark() { }
}

Dog dog = new Dog();
```

**메모리 상태:**
```
[Heap 영역]
┌─────────────────────┐
│ Dog 객체            │
├─────────────────────┤
│ Animal 부분:        │
│   name = "동물"     │
│   eat()            │
├─────────────────────┤
│ Dog 부분:           │
│   breed = "진돗개"  │
│   bark()           │
└─────────────────────┘

dog → 객체 전체 참조
```

## 실생활 비유로 이해하기

### 비유 1: 가족 유전 🧬

**상황:** 부모의 특징을 자식이 물려받는 것

```
부모 (Animal)
- 이름
- 나이
- 먹기
- 자기

↓ 상속

자식 (Dog)
- 이름 (상속)
- 나이 (상속)
- 먹기 (상속)
- 자기 (상속)
- 품종 (추가)
- 짖기 (추가)
```

**코드로 표현:**
```java
public class Parent {
    String name;
    String bloodType;
    int height;

    public void walk() {
        System.out.println(name + "이(가) 걷습니다.");
    }

    public void talk() {
        System.out.println(name + "이(가) 말합니다.");
    }
}

public class Child extends Parent {
    String school;
    int grade;

    // 부모로부터 상속: name, bloodType, height, walk(), talk()

    // 자식만의 특성
    public void study() {
        System.out.println(name + "이(가) " + school + "에서 공부합니다.");
    }

    public void play() {
        System.out.println(name + "이(가) 놉니다.");
    }
}

// 사용
public class FamilyTest {
    public static void main(String[] args) {
        Child child = new Child();
        child.name = "철수";
        child.bloodType = "A";    // 부모로부터 상속
        child.height = 150;       // 부모로부터 상속
        child.school = "행복초등학교";
        child.grade = 3;

        child.walk();   // 부모 메서드
        child.talk();   // 부모 메서드
        child.study();  // 자식 메서드
        child.play();   // 자식 메서드
    }
}
```

**출력:**
```
철수이(가) 걷습니다.
철수이(가) 말합니다.
철수이(가) 행복초등학교에서 공부합니다.
철수이(가) 놉니다.
```

### 비유 2: 직업 계층 구조 👔

**상황:** 회사의 직급 구조

```
Employee (직원)
  ↓
Manager (관리자) - 직원 + 팀 관리 기능
  ↓
Director (이사) - 관리자 + 부서 관리 기능
  ↓
CEO (대표) - 이사 + 회사 전략 기능
```

**코드로 표현:**
```java
// 기본 직원
public class Employee {
    String name;
    String employeeId;
    int salary;

    public Employee(String name, String employeeId, int salary) {
        this.name = name;
        this.employeeId = employeeId;
        this.salary = salary;
    }

    public void work() {
        System.out.println(name + " 직원이 업무를 수행합니다.");
    }

    public void attendMeeting() {
        System.out.println(name + "이(가) 회의에 참석합니다.");
    }
}

// 관리자 (직원 + 팀 관리)
public class Manager extends Employee {
    String team;
    int teamSize;

    public Manager(String name, String employeeId, int salary, String team, int teamSize) {
        super(name, employeeId, salary);  // 부모 생성자 호출
        this.team = team;
        this.teamSize = teamSize;
    }

    // 관리자만의 기능
    public void manageTeam() {
        System.out.println(name + " 관리자가 " + team + " 팀(" + teamSize + "명)을 관리합니다.");
    }

    public void evaluatePerformance() {
        System.out.println(name + "이(가) 팀원 평가를 진행합니다.");
    }
}

// 이사 (관리자 + 부서 관리)
public class Director extends Manager {
    String department;
    int budget;

    public Director(String name, String employeeId, int salary,
                   String team, int teamSize, String department, int budget) {
        super(name, employeeId, salary, team, teamSize);
        this.department = department;
        this.budget = budget;
    }

    // 이사만의 기능
    public void manageBudget() {
        System.out.println(name + " 이사가 " + department + " 부서 예산 " +
                          budget + "억원을 관리합니다.");
    }

    public void makeDecision() {
        System.out.println(name + " 이사가 중요한 의사결정을 합니다.");
    }
}

// CEO (이사 + 회사 전략)
public class CEO extends Director {
    String companyVision;

    public CEO(String name, String employeeId, int salary,
              String team, int teamSize, String department,
              int budget, String companyVision) {
        super(name, employeeId, salary, team, teamSize, department, budget);
        this.companyVision = companyVision;
    }

    // CEO만의 기능
    public void setCompanyDirection() {
        System.out.println("CEO " + name + "이(가) 회사 비전을 제시합니다: " + companyVision);
    }

    public void boardMeeting() {
        System.out.println("CEO가 이사회 회의를 주재합니다.");
    }
}

// 사용
public class CompanyTest {
    public static void main(String[] args) {
        System.out.println("=== 회사 조직도 ===\n");

        Employee emp = new Employee("김사원", "E001", 3000);
        emp.work();
        emp.attendMeeting();
        System.out.println();

        Manager mgr = new Manager("이팀장", "M001", 5000, "개발팀", 10);
        mgr.work();           // Employee로부터 상속
        mgr.manageTeam();     // Manager 고유 기능
        mgr.evaluatePerformance();
        System.out.println();

        Director dir = new Director("박이사", "D001", 8000, "개발팀", 10, "IT부서", 100);
        dir.work();           // Employee로부터 상속
        dir.manageTeam();     // Manager로부터 상속
        dir.manageBudget();   // Director 고유 기능
        dir.makeDecision();
        System.out.println();

        CEO ceo = new CEO("최대표", "C001", 15000, "경영팀", 5, "경영부서", 1000,
                         "글로벌 1위 기업");
        ceo.work();           // Employee로부터 상속
        ceo.manageTeam();     // Manager로부터 상속
        ceo.manageBudget();   // Director로부터 상속
        ceo.setCompanyDirection();  // CEO 고유 기능
        ceo.boardMeeting();
    }
}
```

**출력:**
```
=== 회사 조직도 ===

김사원 직원이 업무를 수행합니다.
김사원이(가) 회의에 참석합니다.

이팀장 직원이 업무를 수행합니다.
이팀장 관리자가 개발팀 팀(10명)을 관리합니다.
이팀장이(가) 팀원 평가를 진행합니다.

박이사 직원이 업무를 수행합니다.
박이사 관리자가 개발팀 팀(10명)을 관리합니다.
박이사 이사가 IT부서 부서 예산 100억원을 관리합니다.
박이사 이사가 중요한 의사결정을 합니다.

최대표 직원이 업무를 수행합니다.
최대표 관리자가 경영팀 팀(5명)을 관리합니다.
최대표 이사가 경영부서 부서 예산 1000억원을 관리합니다.
CEO 최대표이(가) 회사 비전을 제시합니다: 글로벌 1위 기업
CEO가 이사회 회의를 주재합니다.
```

### 비유 3: 전자제품 제품군 📱

**상황:** 스마트폰 제품 라인업

```
SmartDevice (스마트 기기)
  ├─ Smartphone (스마트폰)
  │   ├─ FlagshipPhone (플래그십)
  │   └─ BudgetPhone (보급형)
  ├─ Tablet (태블릿)
  └─ Smartwatch (스마트워치)
```

**코드로 표현:**
```java
// 기본 스마트 기기
public class SmartDevice {
    String brand;
    String model;
    int price;
    int battery;

    public SmartDevice(String brand, String model, int price) {
        this.brand = brand;
        this.model = model;
        this.price = price;
        this.battery = 100;
    }

    public void powerOn() {
        System.out.println(model + " 전원을 켭니다.");
    }

    public void powerOff() {
        System.out.println(model + " 전원을 끕니다.");
    }

    public void charge() {
        battery = 100;
        System.out.println(model + " 충전 완료: " + battery + "%");
    }
}

// 스마트폰
public class Smartphone extends SmartDevice {
    String os;
    int cameraMP;

    public Smartphone(String brand, String model, int price, String os, int cameraMP) {
        super(brand, model, price);
        this.os = os;
        this.cameraMP = cameraMP;
    }

    public void makeCall(String number) {
        System.out.println(model + "에서 " + number + "로 전화를 겁니다.");
    }

    public void takePicture() {
        System.out.println(cameraMP + "MP 카메라로 사진을 촬영합니다.");
    }

    public void installApp(String appName) {
        System.out.println(os + " 앱 '" + appName + "'을(를) 설치합니다.");
    }
}

// 플래그십 스마트폰
public class FlagshipPhone extends Smartphone {
    boolean has5G;
    boolean hasFoldableScreen;

    public FlagshipPhone(String brand, String model, int price,
                        String os, int cameraMP, boolean has5G) {
        super(brand, model, price, os, cameraMP);
        this.has5G = has5G;
        this.hasFoldableScreen = false;
    }

    public void enableAI() {
        System.out.println("AI 기능을 활성화합니다.");
    }

    public void wireless Charging() {
        System.out.println("무선 충전을 시작합니다.");
    }
}

// 태블릿
public class Tablet extends SmartDevice {
    double screenSize;
    boolean hasStylusPen;

    public Tablet(String brand, String model, int price, double screenSize) {
        super(brand, model, price);
        this.screenSize = screenSize;
        this.hasStylusPen = false;
    }

    public void drawWithPen() {
        if (hasStylusPen) {
            System.out.println("펜으로 그림을 그립니다.");
        } else {
            System.out.println("펜이 없습니다.");
        }
    }

    public void splitScreen() {
        System.out.println(screenSize + "인치 화면을 2개로 분할합니다.");
    }
}

// 스마트워치
public class Smartwatch extends SmartDevice {
    boolean hasHeartRateMonitor;
    boolean isWaterproof;

    public Smartwatch(String brand, String model, int price) {
        super(brand, model, price);
        this.hasHeartRateMonitor = true;
        this.isWaterproof = true;
    }

    public void measureHeartRate() {
        System.out.println("심박수를 측정합니다: 75 bpm");
    }

    public void trackExercise(String exerciseType) {
        System.out.println(exerciseType + " 운동을 추적합니다.");
    }
}

// 사용
public class DeviceTest {
    public static void main(String[] args) {
        System.out.println("=== 스마트 기기 테스트 ===\n");

        // 플래그십 폰
        FlagshipPhone flagship = new FlagshipPhone("Samsung", "Galaxy S24", 1200000,
                                                    "Android", 200, true);
        flagship.powerOn();          // SmartDevice로부터
        flagship.makeCall("010-1234-5678");  // Smartphone로부터
        flagship.takePicture();      // Smartphone로부터
        flagship.enableAI();         // FlagshipPhone 고유
        flagship.wirelessCharging(); // FlagshipPhone 고유
        System.out.println();

        // 태블릿
        Tablet tablet = new Tablet("Apple", "iPad Pro", 1500000, 12.9);
        tablet.powerOn();            // SmartDevice로부터
        tablet.splitScreen();        // Tablet 고유
        tablet.hasStylusPen = true;
        tablet.drawWithPen();        // Tablet 고유
        System.out.println();

        // 스마트워치
        Smartwatch watch = new Smartwatch("Apple", "Watch Series 9", 600000);
        watch.powerOn();             // SmartDevice로부터
        watch.measureHeartRate();    // Smartwatch 고유
        watch.trackExercise("러닝"); // Smartwatch 고유
    }
}
```

### 비유 4: 교통수단 계층 🚗

**상황:** 다양한 교통수단의 공통점과 차이점

```
Vehicle (탈것)
  ├─ LandVehicle (육상)
  │   ├─ Car (자동차)
  │   ├─ Bicycle (자전거)
  │   └─ Train (기차)
  ├─ AirVehicle (항공)
  │   ├─ Airplane (비행기)
  │   └─ Helicopter (헬기)
  └─ WaterVehicle (해상)
      ├─ Ship (배)
      └─ Submarine (잠수함)
```

**코드로 표현:**
```java
// 기본 탈것
public class Vehicle {
    String name;
    int maxSpeed;
    int capacity;

    public Vehicle(String name, int maxSpeed, int capacity) {
        this.name = name;
        this.maxSpeed = maxSpeed;
        this.capacity = capacity;
    }

    public void start() {
        System.out.println(name + " 시동을 겁니다.");
    }

    public void stop() {
        System.out.println(name + " 정지합니다.");
    }

    public void displayInfo() {
        System.out.println("=== " + name + " ===");
        System.out.println("최고 속도: " + maxSpeed + "km/h");
        System.out.println("탑승 인원: " + capacity + "명");
    }
}

// 육상 교통수단
public class LandVehicle extends Vehicle {
    int wheelCount;

    public LandVehicle(String name, int maxSpeed, int capacity, int wheelCount) {
        super(name, maxSpeed, capacity);
        this.wheelCount = wheelCount;
    }

    public void drive() {
        System.out.println(wheelCount + "개의 바퀴로 주행합니다.");
    }
}

// 자동차
public class Car extends LandVehicle {
    String fuelType;
    int doors;

    public Car(String name, int maxSpeed, int capacity, String fuelType, int doors) {
        super(name, maxSpeed, capacity, 4);  // 자동차는 4바퀴
        this.fuelType = fuelType;
        this.doors = doors;
    }

    public void openTrunk() {
        System.out.println("트렁크를 엽니다.");
    }

    public void honk() {
        System.out.println("경적을 울립니다: 빵빵!");
    }
}

// 항공 교통수단
public class AirVehicle extends Vehicle {
    int maxAltitude;

    public AirVehicle(String name, int maxSpeed, int capacity, int maxAltitude) {
        super(name, maxSpeed, capacity);
        this.maxAltitude = maxAltitude;
    }

    public void takeOff() {
        System.out.println(name + " 이륙합니다.");
    }

    public void land() {
        System.out.println(name + " 착륙합니다.");
    }

    public void fly() {
        System.out.println(maxAltitude + "m 고도로 비행합니다.");
    }
}

// 비행기
public class Airplane extends AirVehicle {
    int flightNumber;
    String destination;

    public Airplane(String name, int maxSpeed, int capacity,
                   int maxAltitude, int flightNumber, String destination) {
        super(name, maxSpeed, capacity, maxAltitude);
        this.flightNumber = flightNumber;
        this.destination = destination;
    }

    public void serveMeal() {
        System.out.println("기내식을 제공합니다.");
    }

    public void announceDestination() {
```

### 비유 5: 계좌 종류 💰

**상황:** 은행 계좌의 다양한 종류

```
BankAccount (은행 계좌)
  ├─ SavingsAccount (예금 계좌)
  │   └─ HighInterestSavings (고금리 예금)
  ├─ CheckingAccount (수표 계좌)
  └─ InvestmentAccount (투자 계좌)
      ├─ StockAccount (주식 계좌)
      └─ FundAccount (펀드 계좌)
```

**코드로 표현:**
```java
// 기본 은행 계좌
public class BankAccount {
    String accountNumber;
    String ownerName;
    long balance;

    public BankAccount(String accountNumber, String ownerName) {
        this.accountNumber = accountNumber;
        this.ownerName = ownerName;
        this.balance = 0;
    }

    public void deposit(long amount) {
        balance += amount;
        System.out.println(amount + "원 입금. 잔액: " + balance + "원");
    }

    public boolean withdraw(long amount) {
        if (balance >= amount) {
            balance -= amount;
            System.out.println(amount + "원 출금. 잔액: " + balance + "원");
            return true;
        }
        System.out.println("잔액 부족!");
        return false;
    }

    public void displayInfo() {
        System.out.println("━━━━━━━━━━━━━━━━━━━━");
        System.out.println("계좌번호: " + accountNumber);
        System.out.println("예금주: " + ownerName);
        System.out.println("잔액: " + balance + "원");
    }
}

// 예금 계좌
public class SavingsAccount extends BankAccount {
    double interestRate;  // 이자율
    int months;           // 예금 기간

    public SavingsAccount(String accountNumber, String ownerName,
                         double interestRate, int months) {
        super(accountNumber, ownerName);
        this.interestRate = interestRate;
        this.months = months;
    }

    public void calculateInterest() {
        long interest = (long)(balance * interestRate * months / 12);
        System.out.println("예상 이자: " + interest + "원");
        System.out.println("만기 금액: " + (balance + interest) + "원");
    }

    @Override
    public void displayInfo() {
        super.displayInfo();
        System.out.println("이자율: " + (interestRate * 100) + "%");
        System.out.println("예금 기간: " + months + "개월");
        System.out.println("━━━━━━━━━━━━━━━━━━━━");
    }
}

// 고금리 예금
public class HighInterestSavings extends SavingsAccount {
    long minimumBalance;  // 최소 유지 금액

    public HighInterestSavings(String accountNumber, String ownerName,
                              double interestRate, int months, long minimumBalance) {
        super(accountNumber, ownerName, interestRate, months);
        this.minimumBalance = minimumBalance;
    }

    @Override
    public boolean withdraw(long amount) {
        if (balance - amount < minimumBalance) {
            System.out.println("출금 불가! 최소 " + minimumBalance + "원 유지 필요");
            return false;
        }
        return super.withdraw(amount);
    }

    public void checkMinimumBalance() {
        if (balance >= minimumBalance) {
            System.out.println("✓ 최소 잔액 조건 충족");
        } else {
            System.out.println("✗ 최소 잔액 미달: " +
                             (minimumBalance - balance) + "원 부족");
        }
    }
}

// 수표 계좌
public class CheckingAccount extends BankAccount {
    int freeWithdrawals;  // 무료 출금 횟수
    int withdrawCount;

    public CheckingAccount(String accountNumber, String ownerName, int freeWithdrawals) {
        super(accountNumber, ownerName);
        this.freeWithdrawals = freeWithdrawals;
        this.withdrawCount = 0;
    }

    @Override
    public boolean withdraw(long amount) {
        withdrawCount++;
        if (withdrawCount > freeWithdrawals) {
            int fee = 1000;
            System.out.println("출금 수수료 " + fee + "원 부과");
            amount += fee;
        }
        return super.withdraw(amount);
    }

    public void writeCheck(String recipient, long amount) {
        System.out.println("수표 발행: " + recipient + "에게 " + amount + "원");
        withdraw(amount);
    }
}

// 투자 계좌
public class InvestmentAccount extends BankAccount {
    String investmentType;
    double returnRate;

    public InvestmentAccount(String accountNumber, String ownerName,
                            String investmentType, double returnRate) {
        super(accountNumber, ownerName);
        this.investmentType = investmentType;
        this.returnRate = returnRate;
    }

    public void invest(long amount) {
        if (balance >= amount) {
            balance -= amount;
            System.out.println(amount + "원 " + investmentType + " 투자");
        } else {
            System.out.println("투자 금액 부족!");
        }
    }

    public void calculateReturn(int years) {
        long expectedReturn = (long)(balance * returnRate * years);
        System.out.println(years + "년 후 예상 수익: " + expectedReturn + "원");
    }
}
```
