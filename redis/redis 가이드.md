## 🛠️ 실습 환경 요구사항

### 최소 요구사항
- **OS**: Windows 10+, macOS 10.15+, Ubuntu 18.04+
- **메모리**: 4GB RAM 이상
- **디스크**: 10GB 여유 공간
- **네트워크**: 인터넷 연결 (패키지 설치용)

### 권장 요구사항
- **메모리**: 8GB RAM 이상
- **CPU**: 4코어 이상
- **도구**: Docker, VS Code, Git

### 필수 소프트웨어
```bash
# 기본 도구
- Redis (최신 안정 버전)
- Docker & Docker Compose
- 코드 에디터 (VS Code 권장)

# 선택 도구 (언어별)
- Python 3.8+ (redis-py)
- Node.js 16+ (node_redis)
- Java 11+ (Jedis/Lettuce)
- .NET Core 3.1+ (StackExchange.Redis)
```
