## 메서드 오버로딩이란 무엇인가?

### 핵심 개념
메서드 오버로딩(Method Overloading)은 **같은 이름의 메서드를 매개변수의 타입, 개수, 순서를 다르게 하여 여러 개 정의하는 것**입니다.

### 기본 구조
```java
public class Calculator {
    // 메서드 1: 정수 2개 더하기
    public int add(int a, int b) {
        return a + b;
    }

    // 메서드 2: 정수 3개 더하기 (매개변수 개수 다름)
    public int add(int a, int b, int c) {
        return a + b + c;
    }

    // 메서드 3: 실수 2개 더하기 (매개변수 타입 다름)
    public double add(double a, double b) {
        return a + b;
    }
}

// 사용
Calculator calc = new Calculator();
System.out.println(calc.add(1, 2));        // 메서드 1 호출: 3
System.out.println(calc.add(1, 2, 3));     // 메서드 2 호출: 6
System.out.println(calc.add(1.5, 2.5));    // 메서드 3 호출: 4.0
```

### 오버로딩 규칙

**✅ 오버로딩 가능 (구분 기준):**
1. **매개변수 개수가 다름**
```java
public void print(String message) { }
public void print(String message, int count) { }
```

2. **매개변수 타입이 다름**
```java
public void print(int number) { }
public void print(String message) { }
```

3. **매개변수 순서가 다름** (타입이 다를 때만)
```java
public void print(int number, String message) { }
public void print(String message, int number) { }
```

**❌ 오버로딩 불가능:**
1. **매개변수 이름만 다름**
```java
public void print(String message) { }
public void print(String text) { }  // ❌ 컴파일 에러
```

2. **반환 타입만 다름**
```java
public int getValue() { }
public String getValue() { }  // ❌ 컴파일 에러
```

3. **접근 제어자만 다름**
```java
public void print(String message) { }
private void print(String message) { }  // ❌ 컴파일 에러
```

### 오버로딩 vs 오버라이딩

| 구분 | 오버로딩 (Overloading) | 오버라이딩 (Overriding) |
|------|------------------------|-------------------------|
| 정의 | 같은 이름, 다른 매개변수 | 상속 관계에서 메서드 재정의 |
| 위치 | 같은 클래스 내 | 부모-자식 클래스 간 |
| 시그니처 | 다름 | 같음 |
| 반환 타입 | 무관 | 같거나 하위 타입 |
| 사용 시점 | 컴파일 타임 | 런타임 |

```java
// 오버로딩
public class Math {
    public int max(int a, int b) { return a > b ? a : b; }
    public double max(double a, double b) { return a > b ? a : b; }
}

// 오버라이딩
class Animal {
    public void sound() { System.out.println("동물 소리"); }
}
class Dog extends Animal {
    @Override
    public void sound() { System.out.println("멍멍"); }
}
```

### 자동 형변환과 오버로딩

```java
public class TypeConversion {
    public void print(int value) {
        System.out.println("int: " + value);
    }

    public void print(long value) {
        System.out.println("long: " + value);
    }

    public void print(double value) {
        System.out.println("double: " + value);
    }
}

TypeConversion tc = new TypeConversion();
tc.print(10);       // int 호출
tc.print(10L);      // long 호출
tc.print(10.0);     // double 호출

byte b = 10;
tc.print(b);        // int 호출 (byte → int 자동 형변환)

short s = 10;
tc.print(s);        // int 호출 (short → int 자동 형변환)
```

**형변환 우선순위:**
```
byte/short/char → int → long → float → double
```

### 가변 인자(Varargs)와 오버로딩

```java
public class VarargsExample {
    // 일반 메서드
    public int sum(int a, int b) {
        return a + b;
    }

    // 가변 인자 메서드
    public int sum(int... numbers) {
        int total = 0;
        for (int num : numbers) {
            total += num;
        }
        return total;
    }
}

VarargsExample ve = new VarargsExample();
System.out.println(ve.sum(1, 2));           // 일반 메서드 호출 (우선순위 높음)
System.out.println(ve.sum(1, 2, 3));        // 가변 인자 메서드 호출
System.out.println(ve.sum(1, 2, 3, 4, 5));  // 가변 인자 메서드 호출
```

**가변 인자 주의사항:**
- 가변 인자는 마지막 매개변수로만 가능
- 가변 인자는 하나만 가능
- 일반 메서드가 가변 인자 메서드보다 우선순위 높음

### 비유 1: 프린터 출력 🖨️

**상황:** 프린터는 다양한 형태의 문서를 출력할 수 있습니다.

```
프린터.출력(문서)
- 텍스트 문서 출력
- 이미지 출력
- PDF 출력
- 여러 페이지 출력

같은 "출력" 기능이지만 입력에 따라 다르게 동작
```

**코드로 표현:**
```java
public class Printer {
    // 텍스트 출력
    public void print(String text) {
        System.out.println("텍스트 출력: " + text);
    }

    // 이미지 출력
    public void print(Image image) {
        System.out.println("이미지 출력: " + image.getWidth() + "x" + image.getHeight());
    }

    // 여러 페이지 출력
    public void print(String[] pages) {
        System.out.println("총 " + pages.length + "페이지 출력");
        for (int i = 0; i < pages.length; i++) {
            System.out.println("  페이지 " + (i+1) + ": " + pages[i]);
        }
    }

    // 컬러 출력
    public void print(String text, boolean isColor) {
        System.out.println((isColor ? "컬러" : "흑백") + " 출력: " + text);
    }

    // 여러 부 출력
    public void print(String text, int copies) {
        System.out.println(text + " - " + copies + "부 출력");
    }
}

class Image {
    private int width, height;
    public Image(int w, int h) { width = w; height = h; }
    public int getWidth() { return width; }
    public int getHeight() { return height; }
}

// 사용 예시
public class PrinterTest {
    public static void main(String[] args) {
        Printer printer = new Printer();

        // 다양한 출력 방식
        printer.print("안녕하세요");                          // 텍스트
        printer.print(new Image(1920, 1080));               // 이미지
        printer.print(new String[]{"페이지1", "페이지2"});   // 여러 페이지
        printer.print("보고서", true);                       // 컬러
        printer.print("계약서", 3);                          // 여러 부
    }
}
```

**출력:**
```
텍스트 출력: 안녕하세요
이미지 출력: 1920x1080
총 2페이지 출력
  페이지 1: 페이지1
  페이지 2: 페이지2
컬러 출력: 보고서
계약서 - 3부 출력
```

**핵심:**
- 같은 "print" 메서드
- 다양한 입력 타입 처리
- 사용자 입장에서 간단하고 직관적

### 비유 2: 택배 발송 📦

**상황:** 택배 발송은 다양한 옵션이 있습니다.

```
택배발송(받는사람)                    → 기본 배송
택배발송(받는사람, 주소)              → 주소 지정
택배발송(받는사람, 주소, 배송옵션)   → 빠른 배송, 새벽 배송 등
택배발송(받는사람, 주소, 배송옵션, 결제방법)
```

**코드로 표현:**
```java
public class DeliveryService {
    // 기본 배송 (등록된 기본 주소 사용)
    public String ship(String recipient) {
        String trackingNumber = generateTrackingNumber();
        System.out.println("=== 기본 배송 ===");
        System.out.println("받는 사람: " + recipient);
        System.out.println("배송지: 기본 주소");
        System.out.println("배송 방법: 일반 배송");
        System.out.println("운송장 번호: " + trackingNumber);
        return trackingNumber;
    }

    // 주소 지정 배송
    public String ship(String recipient, String address) {
        String trackingNumber = generateTrackingNumber();
        System.out.println("\n=== 주소 지정 배송 ===");
        System.out.println("받는 사람: " + recipient);
        System.out.println("배송지: " + address);
        System.out.println("배송 방법: 일반 배송");
        System.out.println("운송장 번호: " + trackingNumber);
        return trackingNumber;
    }

    // 배송 옵션 지정
    public String ship(String recipient, String address, String deliveryOption) {
        String trackingNumber = generateTrackingNumber();
        int fee = calculateFee(deliveryOption);

        System.out.println("\n=== 옵션 배송 ===");
        System.out.println("받는 사람: " + recipient);
        System.out.println("배송지: " + address);
        System.out.println("배송 방법: " + deliveryOption);
        System.out.println("추가 요금: " + fee + "원");
        System.out.println("운송장 번호: " + trackingNumber);
        return trackingNumber;
    }

    // 배송 옵션 + 결제 방법
    public String ship(String recipient, String address, String deliveryOption,
                      String paymentMethod) {
        String trackingNumber = generateTrackingNumber();
        int fee = calculateFee(deliveryOption);

        System.out.println("\n=== 완전 맞춤 배송 ===");
        System.out.println("받는 사람: " + recipient);
        System.out.println("배송지: " + address);
        System.out.println("배송 방법: " + deliveryOption);
        System.out.println("결제 방법: " + paymentMethod);
        System.out.println("추가 요금: " + fee + "원");
        System.out.println("운송장 번호: " + trackingNumber);
        return trackingNumber;
    }

    // 대량 발송 (여러 수령인)
    public String[] ship(String[] recipients, String address) {
        System.out.println("\n=== 대량 발송 ===");
        System.out.println("수령인 수: " + recipients.length + "명");
        System.out.println("공통 배송지: " + address);

        String[] trackingNumbers = new String[recipients.length];
        for (int i = 0; i < recipients.length; i++) {
            trackingNumbers[i] = generateTrackingNumber();
            System.out.println("  " + (i+1) + ". " + recipients[i] +
                              " - " + trackingNumbers[i]);
        }
        return trackingNumbers;
    }

    private String generateTrackingNumber() {
        return "TRK" + System.currentTimeMillis() % 1000000;
    }

    private int calculateFee(String option) {
        switch (option) {
            case "새벽배송": return 3000;
            case "당일배송": return 5000;
            case "퀵서비스": return 10000;
            default: return 0;
        }
    }
}

// 사용 예시
public class DeliveryTest {
    public static void main(String[] args) {
        DeliveryService delivery = new DeliveryService();

        // 다양한 발송 방식
        delivery.ship("김철수");
        delivery.ship("이영희", "서울시 강남구");
        delivery.ship("박민수", "부산시 해운대구", "새벽배송");
        delivery.ship("최지은", "대구시 수성구", "당일배송", "카드");
        delivery.ship(new String[]{"홍길동", "김유신"}, "회사 주소");
    }
}
```

**출력:**
```
=== 기본 배송 ===
받는 사람: 김철수
배송지: 기본 주소
배송 방법: 일반 배송
운송장 번호: TRK123456

=== 주소 지정 배송 ===
받는 사람: 이영희
배송지: 서울시 강남구
배송 방법: 일반 배송
운송장 번호: TRK123457

=== 옵션 배송 ===
받는 사람: 박민수
배송지: 부산시 해운대구
배송 방법: 새벽배송
추가 요금: 3000원
운송장 번호: TRK123458

=== 완전 맞춤 배송 ===
받는 사람: 최지은
배송지: 대구시 수성구
배송 방법: 당일배송
결제 방법: 카드
추가 요금: 5000원
운송장 번호: TRK123459

=== 대량 발송 ===
수령인 수: 2명
공통 배송지: 회사 주소
  1. 홍길동 - TRK123460
  2. 김유신 - TRK123461
```

**핵심:**
- 같은 "ship" 메서드
- 필요한 정보만 입력
- 점진적으로 세부 옵션 추가

### 비유 3: 커피 주문 ☕

**상황:** 카페에서 커피를 주문하는 다양한 방법

```
주문("아메리카노")                     → 기본 사이즈, 기본 옵션
주문("아메리카노", "라지")             → 사이즈 지정
주문("아메리카노", "라지", true)       → 얼음 추가
주문("아메리카노", "라지", true, 2)    → 샷 추가
```

**코드로 표현:**
```java
public class CoffeeShop {
    // 기본 주문
    public void order(String coffeeName) {
        System.out.println("\n━━━━━━━━━━━━━━━━━━━━━━");
        System.out.println("☕ " + coffeeName);
        System.out.println("사이즈: 미디엄");
        System.out.println("온도: HOT");
        System.out.println("샷: 기본 (1샷)");
        System.out.println("가격: 4,000원");
        System.out.println("━━━━━━━━━━━━━━━━━━━━━━");
    }

    // 사이즈 지정 주문
    public void order(String coffeeName, String size) {
        int price = calculatePrice(size);
        System.out.println("\n━━━━━━━━━━━━━━━━━━━━━━");
        System.out.println("☕ " + coffeeName);
        System.out.println("사이즈: " + size);
        System.out.println("온도: HOT");
        System.out.println("샷: 기본 (1샷)");
        System.out.println("가격: " + String.format("%,d", price) + "원");
        System.out.println("━━━━━━━━━━━━━━━━━━━━━━");
    }

    // 사이즈 + 아이스 주문
    public void order(String coffeeName, String size, boolean isIced) {
        int price = calculatePrice(size);
        System.out.println("\n━━━━━━━━━━━━━━━━━━━━━━");
        System.out.println("☕ " + coffeeName);
        System.out.println("사이즈: " + size);
        System.out.println("온도: " + (isIced ? "ICE" : "HOT"));
        System.out.println("샷: 기본 (1샷)");
        System.out.println("가격: " + String.format("%,d", price) + "원");
        System.out.println("━━━━━━━━━━━━━━━━━━━━━━");
    }

    // 완전 맞춤 주문
    public void order(String coffeeName, String size, boolean isIced, int extraShots) {
        int price = calculatePrice(size) + (extraShots * 500);
        int totalShots = 1 + extraShots;

        System.out.println("\n━━━━━━━━━━━━━━━━━━━━━━");
        System.out.println("☕ " + coffeeName);
        System.out.println("사이즈: " + size);
        System.out.println("온도: " + (isIced ? "ICE" : "HOT"));
        System.out.println("샷: " + totalShots + "샷 (추가 " + extraShots + "샷)");
        System.out.println("가격: " + String.format("%,d", price) + "원");
        System.out.println("━━━━━━━━━━━━━━━━━━━━━━");
    }

    // 대량 주문 (회사 단체 주문)
    public void order(String coffeeName, int quantity) {
        int totalPrice = 4000 * quantity;
        System.out.println("\n━━━━━━━━━━━━━━━━━━━━━━");
        System.out.println("☕ " + coffeeName + " x " + quantity + "잔");
        System.out.println("사이즈: 미디엄");
        System.out.println("온도: HOT");
        System.out.println("총 가격: " + String.format("%,d", totalPrice) + "원");
        System.out.println("━━━━━━━━━━━━━━━━━━━━━━");
    }

    private int calculatePrice(String size) {
        switch (size) {
            case "스몰": return 3500;
            case "미디엄": return 4000;
            case "라지": return 4500;
            case "벤티": return 5000;
            default: return 4000;
        }
    }
}

// 사용 예시
public class CoffeeShopTest {
    public static void main(String[] args) {
        CoffeeShop shop = new CoffeeShop();

        System.out.println("╔══════════════════════════╗");
        System.out.println("║   카페 주문 시스템       ║");
        System.out.println("╚══════════════════════════╝");

        shop.order("아메리카노");
        shop.order("카페라떼", "라지");
        shop.order("카푸치노", "벤티", true);
        shop.order("에스프레소", "미디엄", false, 2);
        shop.order("아메리카노", 10);  // 회사 단체 주문
    }
}
```

### 비유 4: 검색 기능 🔍

**상황:** 웹사이트 검색 기능의 다양한 옵션

```
검색("자바")                          → 전체 검색
검색("자바", "제목")                  → 제목에서만 검색
검색("자바", "제목", 최신순)          → 정렬 옵션
검색("자바", "제목", 최신순, 10개)    → 결과 개수 제한
```

**코드로 표현:**
```java
public class SearchEngine {
    private String[] database = {
        "자바의 정석 - 남궁성",
        "이펙티브 자바 - 조슈아 블로크",
        "모던 자바 인 액션 - 라울-게이브리얼 우르마",
        "자바 프로그래밍 입문 - 김철수",
        "스프링 부트와 AWS로 혼자 구현하는 웹 서비스",
        "코딩 인터뷰 완전 분석 - 게일 맥도웰"
    };

    // 기본 검색 (전체 검색)
    public void search(String keyword) {
        System.out.println("\n━━━━━━━━━━━━━━━━━━━━━━━━━━");
        System.out.println("🔍 검색어: " + keyword);
        System.out.println("검색 범위: 전체");
        System.out.println("정렬: 관련도순");
        System.out.println("━━━━━━━━━━━━━━━━━━━━━━━━━━");

        int count = 0;
        for (String item : database) {
            if (item.contains(keyword)) {
                System.out.println("  ✓ " + item);
                count++;
            }
        }
        System.out.println("총 " + count + "건 검색");
    }

    // 범위 지정 검색
    public void search(String keyword, String searchType) {
        System.out.println("\n━━━━━━━━━━━━━━━━━━━━━━━━━━");
        System.out.println("🔍 검색어: " + keyword);
        System.out.println("검색 범위: " + searchType);
        System.out.println("정렬: 관련도순");
        System.out.println("━━━━━━━━━━━━━━━━━━━━━━━━━━");

        int count = 0;
        for (String item : database) {
            boolean match = false;
            if (searchType.equals("제목") && item.split(" - ")[0].contains(keyword)) {
                match = true;
            } else if (searchType.equals("저자") && item.split(" - ").length > 1 &&
                      item.split(" - ")[1].contains(keyword)) {
                match = true;
            } else if (searchType.equals("전체") && item.contains(keyword)) {
                match = true;
            }

            if (match) {
                System.out.println("  ✓ " + item);
                count++;
            }
        }
        System.out.println("총 " + count + "건 검색");
    }

    // 범위 + 정렬 검색
    public void search(String keyword, String searchType, String sortType) {
        System.out.println("\n━━━━━━━━━━━━━━━━━━━━━━━━━━");
        System.out.println("🔍 검색어: " + keyword);
        System.out.println("검색 범위: " + searchType);
        System.out.println("정렬: " + sortType);
        System.out.println("━━━━━━━━━━━━━━━━━━━━━━━━━━");

        int count = 0;
        for (String item : database) {
            if (item.contains(keyword)) {
                System.out.println("  ✓ " + item);
                count++;
            }
        }
        System.out.println("총 " + count + "건 검색 (" + sortType + " 정렬됨)");
    }

    // 완전 맞춤 검색 (결과 개수 제한)
    public void search(String keyword, String searchType, String sortType, int maxResults) {
        System.out.println("\n━━━━━━━━━━━━━━━━━━━━━━━━━━");
        System.out.println("🔍 검색어: " + keyword);
        System.out.println("검색 범위: " + searchType);
        System.out.println("정렬: " + sortType);
        System.out.println("최대 결과: " + maxResults + "개");
        System.out.println("━━━━━━━━━━━━━━━━━━━━━━━━━━");

        int count = 0;
        for (String item : database) {
            if (item.contains(keyword)) {
                System.out.println("  ✓ " + item);
                count++;
                if (count >= maxResults) break;
            }
        }
        System.out.println("총 " + count + "건 표시 (상위 " + maxResults + "개)");
    }

    // 다중 키워드 검색
    public void search(String[] keywords) {
        System.out.println("\n━━━━━━━━━━━━━━━━━━━━━━━━━━");
        System.out.print("🔍 검색어: ");
        for (String kw : keywords) {
            System.out.print(kw + " ");
        }
        System.out.println("\n검색 범위: 전체");
        System.out.println("조건: OR (하나라도 포함)");
        System.out.println("━━━━━━━━━━━━━━━━━━━━━━━━━━");

        int count = 0;
        for (String item : database) {
            boolean match = false;
            for (String keyword : keywords) {
                if (item.contains(keyword)) {
                    match = true;
                    break;
                }
            }
            if (match) {
                System.out.println("  ✓ " + item);
                count++;
            }
        }
        System.out.println("총 " + count + "건 검색");
    }
}
```

### 비유 5: 메시지 전송 📨

**상황:** 메시지 앱의 다양한 전송 방식

```
전송(메시지)                        → 단순 텍스트
전송(메시지, 수신자)                → 특정인에게 전송
전송(메시지, 수신자목록)            → 여러 명에게 전송
전송(메시지, 수신자, 예약시간)      → 예약 전송
전송(이미지파일)                    → 이미지 전송
전송(파일)                          → 파일 전송
```

**코드로 표현:**
```java
public class MessageApp {
    // 단순 텍스트 (나에게 보내기)

        System.out.println("내용: " + message);

        System.out.println("\n━━━━━━━━━━━━━━━━━━━━━━");
        System.out.println("📨 메시지 전송");
        System.out.println("받는 사람: " + recipient);
        System.out.println("내용: " + message);
        System.out.println("발송 시간: 즉시");
```

## 정리

### 메서드 오버로딩의 핵심

**1. 왜 사용하는가?**
- 같은 기능을 다양한 입력으로 처리
- 메서드 이름 통일로 가독성 향상
